# Fake TRP Detection System
## Enterprise AI-Based Television Rating Fraud Detection
This project detects fake TRP (Television Rating Point) manipulation using Machine Learning and behavioral analytics.
---
## Features
- AI-based fraud prediction
- Random Forest Machine Learning model
- Secure login system
- Streamlit enterprise dashboard
- Fraud analytics visualization
- SQLite database integration
- Behavioral pattern analysis
- Real-time prediction system
- Advanced fraud dataset support
---
## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- SQLite
- Matplotlib
- Bcrypt
---
## Dataset Features
The system uses advanced TRP behavioral features including:
- viewing_minutes
- devices_used
- location_count
- trp_value
- device_id
- ip_address
- session_duration
- channel_switches
- watch_pattern_score
- device_behavior_score
- region
- time_of_day
---
## Machine Learning Model
Algorithm Used:
- Random Forest Classifier
The model predicts whether TRP activity is:
- Genuine TRP
- Fake TRP
